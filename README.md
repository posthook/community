# Posthook Community

Welcome to the Posthook community! This is the place to report bugs, request features, ask questions, and see what we're working on.

## Quick Links

- [Documentation](https://docs.posthook.io)
- [Changelog](https://docs.posthook.io/changelog)
- [Status Page](https://status.posthook.io)

## How to Participate

### Report a Bug

Found something broken? [Open a bug report](https://github.com/posthook/community/issues/new?template=bug_report.yml).

Please include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your environment (language/SDK, if applicable)

### Request a Feature

Have an idea? [Start a discussion](https://github.com/posthook/community/discussions/new?category=feature-requests).

Before posting, please search existing discussions to avoid duplicates. Use the 👍 reaction to vote on features you'd like to see.

### Ask a Question

Stuck on something? [Ask in Q&A](https://github.com/posthook/community/discussions/new?category=q-a).

### Share What You've Built

Using Posthook in a cool way? [Show it off](https://github.com/posthook/community/discussions/new?category=show-and-tell)!

## Code of Conduct

We're committed to providing a welcoming and inclusive experience for everyone. Please be respectful and constructive in all interactions.

## Security

If you discover a security vulnerability, please email security@posthook.io instead of opening a public issue.
